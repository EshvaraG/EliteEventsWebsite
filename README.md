Elite Events Website – README

Overview

The Elite Events Website is a responsive event planning website developed using HTML, CSS, and JavaScript. It allows users to explore event planning services, browse event galleries, view available packages, and submit booking requests through an online booking form.

Setup
Download or clone the project folder.
Open the project in Visual Studio Code (or any code editor).
Ensure all HTML files, CSS files, JavaScript files, and images remain in their correct folders.
Install the Live Server extension in Visual Studio Code (recommended).
Right-click index.html and select Open with Live Server.
The website will open in your web browser.

Architecture
The project follows a simple front-end architecture:

HTML – Structures all web pages.
CSS – Controls styling, layout, colours, and responsiveness.
JavaScript – Handles interactive features such as the booking form, event selection, alerts, and page navigation.
Images Folder – Stores all event and gallery images.
Multiple HTML Pages – Separate pages for each event type, packages, locations, and social media links.

Known Limitations
No user registration or login functionality.
Booking requests are not stored in a database.
No online payment system.
Contact and newsletter forms do not send emails.
Booking confirmation is displayed as a browser alert only.
Social media pages are placeholders and not linked to real accounts.
Company addresses, phone numbers, and locations are fictional for demonstration purposes.
The website requires an internet connection to load external resources such as Font Awesome icons and Google Fonts.

This version serves as a functional front-end prototype and can be extended with a back-end database, authentication, and payment integration in future development.
